### Code Contributions

1. **Fork the repository**
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Test thoroughly**
5. **Commit**: `git commit -m 'Add some feature'`
6. **Push**: `git push origin feature/your-feature-name`
7. **Create a Pull Request**
